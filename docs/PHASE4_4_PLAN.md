```
└─────────────────────────────────────────────────────────────┘
                              │
                              │ (boundary)
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                   UNTRUSTED LAYER                           │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  gate_precheck.ts          (UX — bypassable)        │   │
│  │  Client-side cache         (Stale — manipulable)    │   │
│  │  User input                (Hostile — validated)    │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 🚨 Preflight Warning

**`integrations/aa/gate_precheck.ts` is UNTRUSTED.**

It is a UX convenience layer, NOT a security mechanism.

An attacker who bypasses preflight **MUST NOT** be able to bypass the law.

**Enforcement (on-chain or trusted backend) is the SOLE source of truth.**

### Attack Scenarios

| Attack | Mitigation |
|--------|------------|
| Client skips preflight | Enforcement re-validates everything |
| Client spoofs cached signal | Enforcement uses trusted state |
| Client manipulates magnitude | Enforcement applies hard-coded bounds |
| Client sends malformed input | Pack validation rejects (no-throw) |

## Wire Protocol

All external communication uses canonical wire format:

```typescript
// Request (validated by pack_canonical_input.ts)
{
  subject_id: Uint8Array(32),
  action_type: "TRANSFER" | "EXECUTE" | "DEPLOY" | "WRITE",
  scope_hash: Uint8Array(32),
  magnitude: bigint,
  context_min: Uint8Array(32)
}

// Response (generated by reason_wire.ts)
{ verdict: "ALLOW" }
// or
{ verdict: "DENY", code: "INV_OUT_OF_BOUNDS" }
```

**Rules:**
- No `message`, `description`, or `error` fields
- No stack traces
- No internal enum names (use wire format only)
